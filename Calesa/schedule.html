CCNA 213 – Introduction to Networks (Prof. Ryan M. Aguilos)

-This subject introduces you to how computer networks operate, including the architecture, structure, and function of network components such as routers, switches, and cables. You'll study the OSI and TCP/IP models, IP addressing, basic configurations, and use Cisco Packet Tracer to simulate real network environments. By the end, you’ll be able to set up simple local networks and troubleshoot basic connectivity issues.

IT 233 – Object-Oriented Programming (Prof. Christian Dave S. Ergina)

-This course focuses on the principles of object-oriented programming such as encapsulation, inheritance, abstraction, and polymorphism. You'll build applications by organizing code into classes and objects, and use real-life modeling to write reusable, scalable programs. Typically, this is done using Java or Python depending on the instructor’s preference.

IT 213 – Data Structures and Algorithms (Prof. Eric O. Sta. Singh)

-DSA teaches you how to organize, store, and manipulate data efficiently using arrays, stacks, queues, linked lists, trees, and graphs. You'll also explore algorithms for sorting, searching, and recursion while learning how to analyze their time and space complexity. This subject is foundational for solving programming problems and technical interviews.

IT 253 – Platform Technologies (Prof. Lyra K. Nuevas)

-This subject explores different computing platforms like desktops, mobile devices, web, and cloud platforms. You'll study how operating systems interact with hardware, virtualization, emulation, and the characteristics of embedded and distributed systems. The goal is to understand how software adapts across different technology stacks and devices.

IT 273 – Web Systems and Technologies 1 (Prof. Ronnie G. Cabillan)

-This course introduces front-end web development using HTML, CSS, and JavaScript, focusing on building static and dynamic websites. You’ll learn how browsers render pages, handle user interactions, and manage responsive design. Expect hands-on practice with web page layout, styling, and interactivity.

IT 293 – Statistics and Probability (Prof. Mark Ignacius G. Tobias)

-You’ll learn how to describe, analyze, and interpret data using statistical tools such as mean, median, mode, standard deviation, and probability distributions. Topics also include hypothesis testing, correlation, regression, and sampling techniques. It’s especially useful for data analysis in research and IT fields like machine learning or UX research.

PE 212 – PATHFIT 3 (Team Sports)

-This class develops your teamwork, coordination, and physical fitness through structured team sports activities such as basketball, volleyball, or soccer. You’ll learn basic rules, strategies, and the value of communication and sportsmanship. The class also promotes leadership and healthy lifestyle habits through group-based physical activities.

MRI33 – Life and Works of Rizal (Prof. Kay Bathala H. Santos)

-This subject explores the biography, major writings (like Noli Me Tangere and El Filibusterismo), and historical context of José Rizal's life. You’ll analyze how Rizal's ideas influenced the Philippine revolution and national identity, and reflect on how his advocacy for education, freedom, and justice remains relevant today. The course often includes essays, discussions, and creative presentations.
